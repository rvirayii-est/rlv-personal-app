# RLViray UI

A modern landing page built with Vue 3, TypeScript, and Tailwind CSS.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Build

Build for production:

```bash
npm run build
```

## Preview

Preview the production build:

```bash
npm run preview
```

## Tech Stack

- Vue 3
- TypeScript
- Tailwind CSS
- Vite
